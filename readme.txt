Запуск осуществляется с помощью gradlew.bat run
Либо собрать с помощью gradlew.bat jar и запустить с помощью java -jar <имя файла>.jar
При запуске есть 2 обязательных параметра.
1) ip адрес для мультикаста. Поддерживаются стандарты IPv4 и IPv6
2) порт для прослушивания
3) Имя адаптера, через который процесс будет взаимодействовать с сетью.
Третий параметр необязателен. Если не указать, адаптер выбирается автоматически.
Параметры вводятся через пробел при запуске программы.
